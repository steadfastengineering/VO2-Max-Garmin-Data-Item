# Cooper Test
Receive and instant VO2 max estimate utilizing the cooper formula on your Garmin Smart Watch. 

Simply add this data field to your running activity, run for a period 12 minutes, and view the estimate.  
This does not interfere with Garmin's more complicated data analysis or their estimates of VO2 max. It is a simple and suprisingly accurate assessment that can be used to supplement Garmin's "black box" approach to computin gour VO2 Max.
